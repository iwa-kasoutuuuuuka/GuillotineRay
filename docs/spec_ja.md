# Guillotine Ray - 製品仕様書 (v1.0.9)

## 1. ソフトウェア概要
OpenCV をベースとした Windows 用高性能スクリーン監視・入力シミュレーションソフトウェア。

## 2. コア仕様
- **認識エンジン**: OpenCV 4.10.0 (\CCoeffNormed\ 方式)
- **入力方式**: Win32 \SendInput\ (MOUSEEVENTF_ABSOLUTE 方式)
- **DPI Aware**: \PerMonitorV2\ 完全対応。
- **権限**: 管理者権限を要求 (\pp.manifest\ 実装済み)。

## 3. アップグレード内容 (v1.0.2 -> v1.0.9)
- **入力制御**: \Cursor.Position\ への依存を廃止し、正規化絶対座標 (0-65535) による \SendInput\ 1パケットでの移動・クリック処理に移行。
- **ライフサイクル**: ウィンドウハンドル生成後の初期化（Loadイベント）に全てのUIロジックを集約し、起動時クラッシュを根絶。
- **UI/UX**: ハイコントラスト配色とダイナミックな多言語切り替えボタンの実装。
