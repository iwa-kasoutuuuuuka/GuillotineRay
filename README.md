- **[GuillotineRay_v1.0.9.zip](https://github.com/iwa-kasoutuuuuuka/GuillotineRay/raw/main/releases/GuillotineRay_v1.0.9.zip)** (Windows 64-bit)

**Guillotine Ray** is a professional-grade, high-performance screen monitoring and precision auto-clicking tool built with C# and OpenCV. Optimized for extreme accuracy and stability.

---

## 🚀 Version 1.0.9 Key Features

- **Ultra-High Precision**: New normalized coordinate system (0-65535) for sub-pixel accuracy.
- **Improved UI Visibility**: Enhanced contrast dark-mode interface for better daytime/nighttime usage.
- **One-Click Stability**: Integrated VC++ Redistributable installer included in the package.
- **OpenCV Pattern Matching**: High-accuracy detection using \CCoeffNormed\ algorithm.
- **DPI & Multi-Monitor**: Fully compatible with PerMonitorV2 scaling and virtual desktops.
- **Low Latency**: Performance optimized with ROI-limited capture and grayscale processing.
- **Emergency Stop**: Global hotkey (F8) to instantly halt operations.

## 🇯🇵 日本語概要

**Guillotine Ray** は、C# と OpenCV を駆使して開発された、高性能かつ精密なスクリーン監視型オートクリッカーです。

### 🌟 特徴
- **超精密クリックロジック**: Win32 API の絶対座標系を用いた高い同期性と正確性。
- **高視認性 UI**: 黒背景に白文字のハイコントラストデザインを採用し、操作ミスを防止。
- **簡単セットアップ**: OpenCV の動作に必要なライブラリ（VC++ 再頒布可能パッケージ）を同梱。
- **マルチモニター対応**: 複雑なモニターレイアウトや拡大設定下でも正確に動作。
