# Guillotine Ray 🗡️⚡ (ギロチン・レイ) v1.1.0

![Guillotine Ray Logo](docs/logo.png)

## 📦 ダウンロード
- **[GuillotineRay_v1.1.0.zip](https://github.com/iwa-kasoutuuuuuka/GuillotineRay/raw/main/releases/GuillotineRay_v1.1.0.zip)** (Windows 64-bit)

**Guillotine Ray** は、C# と OpenCV を駆使して開発された、高性能かつ精密なスクリーン監視型オートクリッカーです。

---

## 🌟 主な特徴 (v1.1.0)

- **日本語・英語完全対応**: ボタン一つで UI の表記を切り替え可能。
- **超精密クリック**: Win32 API の絶対座標系を用いた最高精度の入力。
- **高視認性 UI**: 黒背景に白文字のハイコントラストデザイン。
- **簡単セットアップ**: 動作に必要なライブラリ (VC++ Redist) を同梱済み。
- **マルチモニター対応**: スケーリングが異なる環境でも座標を正確に計算。
- **緊急停止機能**: F8 キーで即座に動作を中断。
